Using ReactJS with AngularJS
============================

## Starting up

```
npm install
npm start
open http://localhost:8080/webpack-dev-server/
```

## Versioning

The repository contains tagged versions of each of the steps needed to integrate ReactJS into AngularJS.

To switch use:

```
git checkout v1
git checkout v2
git checkout v3
git checkout v4
git checkout v5
git checkout v6
```
